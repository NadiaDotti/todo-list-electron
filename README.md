# 📝 TODO List - Electron App

## 🚧 Project Status
Work in Progress
Some features are still under development.

## 🚀 Technologies Used
- ElectronJS
- TailwindCSS
- DaisyUI
- JavaScript

## 🛠️ Getting Started
- Clone the repository
```bash
git clone https://github.com/NadiaDotti/todo-list-electron.git
cd todo-list-app
```
- Install dependencies
```bash
npm install
```
- Start the application
```bash
npm start
```
